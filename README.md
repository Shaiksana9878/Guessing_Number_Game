# Number Guessing Game (Java)

## Description
A simple Java console-based Number Guessing Game where the computer generates a random number between 1 and 100.

## Features
- Random number generation
- Hint after every guess
- 5 attempts per round
- Unlimited rounds
- Score tracking (attempt count)
- Continue or quit option

## Technologies Used
- Java
- Scanner Class
- Math.random()

## How to Run

Compile:

```bash
javac NumberGuessingGame.java
```

Run:

```bash
java NumberGuessingGame
```
